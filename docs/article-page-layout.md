### 뜨다 아티클 모바일 레이아웃 정리

#### 1. 전체 화면 구조

- **뷰포트**
  - 기준 해상도: 모바일 세로 \(375 \times 812px\) 정도.
  - 배경색: 흰색.
- **세로 스크롤 단일 페이지**
  - 상단 타이틀, 메인 아티클 카드, 서브 아티클 리스트, 하단 탭 바로 구성.

#### 2. 상단 영역

- **페이지 타이틀**
  - 텍스트: `뜨다 아티클`.
  - 폰트: Pretendard SemiBold, 24px, 블랙.
  - 좌측 정렬, 상단 여백이 넉넉한 단일 헤드라인.

#### 3. 메인 아티클 Large 카드

- **카드 컨테이너**
  - 크기: 약 320 \(\text{w}\) × 360 \(\text{h}\) px.
  - 모서리: 6px 라운드.
  - 전체를 하나의 탭 영역/링크로 사용하는 것이 자연스러운 구조.
- **배경 이미지**
  - 상단 전체를 덮는 대표 이미지.
  - 카드 전체에 적용되며, 하단으로 갈수록 초록색 그라디언트 오버레이가 깔림.
- **그라디언트 오버레이**
  - 위치: 카드 하단 절반 정도.
  - 스타일: 위쪽은 투명, 아래쪽은 진한 그린 색 \(`#0ABE8C` 계열\)으로 부드럽게 변하는 세로 그라디언트.
  - 하단 좌측에 텍스트가 잘 읽히도록 대비 확보 역할.
- **텍스트 영역**
  - 상단: 작성자/역할
    - 예시: `뜨개동산지기 반디`.
    - 폰트: Pretendard Medium, 14px, 흰색.
    - 카드 하단에서 약간 위 위치, 좌측 정렬.
  - 하단: 메인 타이틀
    - 예시: `아티클의 메인 타이틀입니다`.
    - 폰트: Pretendard SemiBold, 20px, 흰색.
    - 한 줄 타이틀 기준, 아래쪽에 더 강조되어 배치.

#### 4. 서브 아티클 Small 카드 리스트

- **리스트 전체**
  - 메인 카드 하단에 세로로 2개 이상 반복.
  - 각 카드 높이: 120px, 너비: 320px.
  - 카드 사이에 적당한 세로 간격.
- **썸네일 영역**
  - 너비: 107px, 높이: 120px.
  - 좌측에 고정 비율 이미지 영역.
  - 일부 카드에서는 다른 이미지 리소스를 혼합해 사용.
- **텍스트 영역**
  - 위치: 썸네일 오른쪽, 약 123px 지점부터 시작.
  - 세로로 두 블록으로 나뉨.
  - **상단 정보 블록**
    - 상단 라벨 배지
      - 예시: `작가 인터뷰`.
      - 배경색: 연한 민트/그린 \(`#66CCB0`\).
      - 둥근 모서리, 좌측 라벨 텍스트 + 우측 동그란 번호 뱃지 구성.
    - 번호 뱃지
      - 동그란 엘립스 아이콘 위에 숫자 \(1, 2 ...\).
      - 숫자 색: 배경색과 동일한 계열 그린, 배경은 흰색 서클.
    - 타이틀
      - 두 줄로 나뉜 메인 타이틀 텍스트.
      - 예시: `아티클의 메인 타이틀은 / 여기에 위치합니다`.
      - 폰트: Pretendard SemiBold, 16px, 진한 회색 \(`#666666`\).
      - 고정 폭 안에서 줄바꿈.
  - **하단 정보 블록**
    - 작성자/역할 텍스트.
    - 예시: `뜨개동산지기 반디`.
    - 폰트: Pretendard Medium, 12–14px 범위, 회색 \(`#666666`\).
    - 카드 맨 아래에 여백을 두고 좌측 정렬.

#### 5. 하단 탭 바

- **탭 바 컨테이너**
  - 높이: 72px.
  - 배경: 흰색.
  - 그림자: `0 0 4px rgba(0, 0, 0, 0.25)` 수준의 얕은 섀도.
  - 좌우 패딩: 36px, 상단 패딩 12px, 하단 패딩 20px.
  - 내부 아이콘들을 가로로 균등 배치.
- **탭 구성**
  - 총 5개 탭: `아티클`, `홈`, `작품`, `피드`, `마이페이지`.
  - 각 탭은 아이콘 + 라벨 텍스트 수직 정렬.
- **선택/비선택 상태**
  - 선택된 탭: `아티클`
    - 아이콘 색: 브랜드 그린 계열.
    - 라벨 텍스트: Pretendard Light, 10px, 그린 색, 중앙 정렬.
  - 비선택 탭들
    - 아이콘 색: 연한 회색.
    - 라벨 텍스트: Pretendard Light, 10px, 회색 \(`#A6A6A6`\).
    - 일부 라벨은 폭이 넓어 중앙 정렬 폭을 넉넉히 사용 \(예: `마이페이지`\).

#### 6. 레이아웃 요약 및 구현 포인트

- **정보 구조**
  - 상단 타이틀 → 대표 Large 카드 → Small 카드 리스트 → 글로벌 하단 네비게이션.
  - 사용자가 진입 시 가장 먼저 메인 아티클을 보고, 이후 나머지 아티클을 탐색하도록 흐름 설계.
- **타이포그래피 계층**
  - 페이지 타이틀 \(24px\) > Large 카드 타이틀 \(20px\) > Small 카드 타이틀 \(16px\) > 보조 텍스트/작성자 \(12–14px\).
  - 컬러 대비: 주요 타이틀은 흰색 또는 짙은 회색, 서브 텍스트는 중간 회색.
- **컬러 톤**
  - 브랜드 메인: 그린/민트 계열 \(`#0ABE8C`, `#66CCB0`\).
  - 중립 계열: 블랙, 진한 회색 \(`#666666`\), 연한 회색 \(`#A6A6A6`\).
  - 카드 배경과 탭 바는 모두 흰색을 사용해 콘텐츠에 집중.
- **반응형 고려 \(확장 아이디어\)**
  - 현재 피그마는 모바일 기준이나, 웹에서는
    - Large 카드를 상단에 크게,
    - Small 카드를 그리드 또는 1열 리스트로 확장 가능.
  - 하단 탭 바는 모바일 전용으로, 데스크톱에서는 상단 내비게이션 바로 재구성 가능.
